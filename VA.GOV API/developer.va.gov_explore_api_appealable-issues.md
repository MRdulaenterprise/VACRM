# Appealable Issues API

benefitsAUTHORIZATION CODE GRANTCLIENT CREDENTIALS GRANT

## With this API you can

- Get a claimant's previously decided issues that may be eligible for appeal.
- Use its data to submit a Higher-Level Review, Notice of Disagreement, or Supplemental Claim.

## Getting access

**Sandbox:** [Start developing](https://developer.va.gov/explore/api/appealable-issues/sandbox-access) immediately with test data.

**Production:** Timeline for getting production access varies. [Learn more about getting production access](https://developer.va.gov/production-access/request-prod-access).
